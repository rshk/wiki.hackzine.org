Sphinx: automatically build documentationÂ¶

This is the small bash script I use to automatically rebuild the Sphinx documentation upon change detected from inotify filesystem watch.

#!/bin/bash

cd "$( dirname "$0" )"

while :; do
    inotifywait -r -e modify,close_write,moved_to,moved_from,move,create,delete ../
    make html
done

Update: A more generic and improved version can be found here: https://github.com/rshk/CommonScripts/blob/master/Linux/Generic/autobuild.sh
